## Featured Projects

### 01 — Comet Tales Prototype
**Role:** Project Manager & Lead UI/UX Designer  
**Timeline:** January – April 2025  
**Platform:** Mobile (iOS / Android) and Web

A high-fidelity educational app combining storytelling and exploration to teach middle school students about astronomical topics. Features exploratory modules, mini-game knowledge checks, interactive AR simulations, and discussion boards.

**Stack:** Figma · Adobe Illustrator · Adobe Photoshop · Adobe Aero  
**Links:** [Live Prototype ↗](https://www.figma.com/proto/gsPqmIZWVUKxvTAiO5MpNq/high-fid-prototype?node-id=105-1048&starting-point-node-id=105%3A1048&scaling=scale-down&content-scaling=fixed&t=Zri9tPH4SnvZldrV-1)

**Retrospective:** User testing showed strong satisfaction and agency over learning. Next steps include full mobile development in React and potential partnership with the Kennedy Space Center.

### 02 — Dust Buddies Prototype
**Role:** UI/UX Designer & Platform Architect  
**Timeline:** Ongoing  
**Platform:** Mobile (iOS / Android)

A gamified roommate chore management app with customizable avatars ("dust buddies"), anonymous voting, group task challenges, and built-in messaging to mediate conflict and encourage accountability.

**Stack:** Figma · Adobe Illustrator · React Native · Firebase · JavaScript  
**Links:** [Live Prototype ↗](https://www.figma.com/proto/7uX3sVdNdCqSU8qQYmBHSv/Hi-fi-Prototype?node-id=6-367&starting-point-node-id=6%3A367&t=NAQlf870cJmXMN6I-1)

**Retrospective:** User feedback confirmed that gamified accountability led to higher task engagement. Currently in active development with an estimated release of April 2026.
